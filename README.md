# MediStore Backend API

A scalable, secure, and production-ready REST API for the MediStore pharmacy management.
Built with modern backend architecture, authentication, and clean code principles.

---

## Project Overview

MediStore Backend is a RESTful API that powers the MediStore frontend application.

It handles:

-  Authentication & Authorization (JWT)
-  User Management
-  Medicine Management
-  Category Management
-  Order Processing
-  Admin Controls
-  Filtering & Searching

The project follows clean architecture principles and scalable folder structure for real-world production usage.

---
